# Roadmaps

Roadmap 按领域管理里程碑，只保留活跃和近期目标。

## 活跃路线图

| 路线图 | 说明 |
|--------|------|
| [mvp-sales-workbench.md](mvp-sales-workbench.md) | **当前 MVP**：YouTube 获客 → 邮件谈价 → WA 辅助；含文档地图与子任务索引 |

## 规则

- 默认路径：`roadmaps/<domain>.md`；
- 单个领域路线图接近 200 行时按里程碑拆分；
- Roadmap 引用 Epic，Epic 引用 Child Change；
- 不在 Roadmap 中复制任务正文或完整历史；
- 已完成内容只保留简短结果链接，旧导航按 `archive/` 规则处理。

模板见 [`../templates/ROADMAP.md`](../templates/ROADMAP.md)。
